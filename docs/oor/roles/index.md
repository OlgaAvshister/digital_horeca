# Слой субъектов: роли участников процесса OOR-IDE

Участники процесса работы с системой управления требованиями OOR-IDE определены тремя ролями. Роли используются для назначения мандатов (прав на действия и переходы состояний); определения терминов - в глоссарии.

| Роль | Назначение | Файл |
|------|------------|------|
| **Analyst** | Проектировщик смыслов | [analyst.md](analyst.md) |
| **AI-Developer** | Исполнитель (автоматизированная роль) | [ai_developer.md](ai_developer.md) |
| **Reviewer** | Контролёр качества и SSOT (Single Source of Truth) | [reviewer.md](reviewer.md) |

## Связь с другими слоями

- **Глоссарий** (`glossary/`) - определяет термины (в т.ч. Mandate, Domain).
- **Домен** (`domains/oor_manager/`) - сущности, переходы и правила ссылаются на роли при назначении мандатов.
- **Сценарии** - описание того, как роли взаимодействуют с интерфейсом OOR-IDE.
